# Build a Hash Table

A hash table is a data structure that stores key-value pairs.  A hash table works by taking the key as the input and then hashing this key according to a specific hashing function.

## Technologies

Uses Python3 - Classes and Dictionaries.

## Deployment

Copy `main.py` into the text editor of your choice.  If using terminal, use the command `python3 main.py` and press the Enter Key.
